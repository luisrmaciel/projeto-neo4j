# Projeto Neo4j - Sistema de Recomendação

Banco de dados de grafo desenvolvido no Neo4j Aura contendo:

- 10 Usuários
- 10 Filmes
- 10 Séries
- 15 Atores
- 10 Diretores
- 8 Gêneros

## Como importar

1. Baixe o arquivo `projeto-neo4j.cypher`
2. Acesse seu Neo4j Browser
3. Copie o conteúdo do arquivo
4. Cole no Neo4j Browser
5. Execute (Ctrl+Enter)
